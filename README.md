# Solar Tracker for Maximum Power

## Overview
This project automatically tracks sunlight using LDR sensors and aligns a solar panel for maximum power generation.

## Components Used
- Arduino Uno
- LDR Sensors
- Servo Motor
- Solar Panel

## Features
- Automatic solar tracking
- Improved energy harvesting
- Real-time panel adjustment
